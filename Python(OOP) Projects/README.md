🏥 Mini Project: Hospital Reliability System
This project demonstrates the implementation of Inheritance and Nested Classes in Python. It focuses on representing a hierarchical relationship between a Hospital, its Doctors, and the underlying System reliability.

🔑 Key Features & Logic:
Class Inheritance: The Doctors class inherits from the Hospital class, allowing it to access parent attributes like Reception and methods like hardworking().

Inner Class (Nested System): The System class is defined inside Hospital, representing a logical grouping where the hospital's performance and reliability are managed as a sub-entity.

Encapsulation of Logic: * Reliability Definition: A custom method that explains the engineering concept of reliability (repeatable results).

Staff Tracking: Managing total staff count through constructor initialization.

Class Attributes: Usage of a boolean flag Reception = True to verify administrative status across the hierarchy.

💻 Code Architecture:
Parent Class: Hospital (Handles staff and general duty).

Child Class: Doctors (Handles specific medical treatments and dues).

Nested Class: System (Handles performance environment and reliability definitions).
